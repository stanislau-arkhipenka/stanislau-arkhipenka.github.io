RC Tank 1

This tank was originally designed by Bryant87 at Thingiverse.

Instead of using standard RC equipment (as suggested in original design) I used raspberry pi zero w to control it over wifi. Doing so allows to add webcam in the future.

BOM

New Parts

  • Raspberry Pi Zero W
  • I2C PWM module
  • DC-DC Converter
  • Heatsink (optional)

Original Parts

  • x2 60A ESC & 540 35T engine
  • x2 Gears
  • x2 Gear adapter
  • x12 Bearing 8x22x7mm 608
  • x12 Bearing 4x13x5mm 624
  • Lipo battery (3 cell)

Bolts

  • x8 M4x15mm
  • x12 M4x60mm
  • x4 M3x20mm
  • x4 M3x10mm
  • x84 M3x45mm

Nuts

  • x88 M3
